Geräte-Wächter

Eigene Überwachungsziele, z.B. die IPTV-Box (Ping) oder der Butler21-Server (mit Port: TCP-Dienst-Check – prüft, ob die Anwendung selbst antwortet). Erscheinen im Dashboard beim Dauerping und im Latenz-Verlauf.

NameHost/IPPort (leer = Ping)

E-Mail-Berichte

Sendet im gewählten Intervall einen Bericht, welche Geräte an Lags/Störungen beteiligt waren. Bei GMX/Web.de/Gmail: SMTP-Server des Anbieters und ggf. App-Passwort verwenden.

Berichte automatisch senden
STARTTLS verwenden (bei Port 465 wird automatisch SSL genutzt)
Bei kritischen Ereignissen sofort mailen (zusätzlich zum Intervall-Bericht)

Programm

Autostart trägt NightmareCatcher in den Windows-Autostart des aktuellen Benutzers ein, sodass der Server nach dem Anmelden automatisch läuft.

Automatisch mit Windows starten

Sicherung

Alle Einstellungen (inklusive Zugangsdaten!) als Datei sichern oder aus einer Sicherung wiederherstellen. Die Datei entsprechend sorgsam aufbewahren.

Geräte & Stockwerke

UniFi-Geräte werden automatisch erkannt und anhand des Namens einem Stockwerk zugeordnet (z.B. "Switch 1.OG" → 1.OG, "AP Büro" → 2.OG). Eine manuelle Auswahl übersteuert die Automatik dauerhaft.
